# Контрольная работа №4

1. Создайте и активируйте виртуальное окружение:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Выполните миграции базы данных (если не выполнены):
   ```bash
   alembic upgrade head
   ```
4. Для запуска приложения:
   ```bash
   uvicorn src.main:app --reload
   ```
   Документация OpenAPI будет доступна по адресу `http://localhost:8000/docs`.

В проекте реализованы модульные асинхронные тесты, которые покрывают основные сценарии (создание, получение, удаление), а также кастомные обработчики ошибок.
Запустить тесты:
```bash
pytest tests/
```
